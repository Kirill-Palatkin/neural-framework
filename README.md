## Описание проекта

Данный проект представляет собой реализацию упрощённого фреймворка для создания и обучения многослойных нейронных сетей. Реализация выполнена на языке Python без использования высокоуровневых библиотек машинного обучения, таких как TensorFlow или PyTorch.

Основной целью проекта является изучение внутренних механизмов работы нейронных сетей, включая прямое распространение сигнала (forward pass), обратное распространение ошибки (backpropagation) и оптимизацию параметров модели.

---

## Функциональные возможности

### 1. Построение нейронных сетей

Фреймворк поддерживает создание многослойных нейронных сетей путём последовательного перечисления слоёв. Реализована возможность гибкой конфигурации архитектуры модели.

Поддерживаемые типы слоёв:
- Полносвязные слои (Dense)

Пример создания модели:
```python
model = Sequential([
    Dense(128, activation="relu"),
    Dense(64, activation="relu"),
    Dense(10, activation="softmax")
])
```

---

### 2. Работа с данными

Реализован набор инструментов для подготовки и обработки данных:

- преобразование данных (`map`)
- перемешивание выборки (`shuffle`)
- разбиение на мини-батчи (`minibatching`)
- формирование пайплайна обработки данных

Данные инструменты позволяют эффективно подготавливать обучающие выборки для процесса обучения модели.

---

### 3. Алгоритмы оптимизации

В рамках проекта реализованы следующие методы оптимизации:

- Стохастический градиентный спуск (SGD)
- SGD с моментом (Momentum SGD)
- Gradient Clipping (ограничение градиента)

---

### 4. Функции потерь

Фреймворк поддерживает функции потерь для различных типов задач:

**Для задач регрессии:**
- Mean Squared Error (MSE)

**Для задач классификации:**
- Cross-Entropy Loss
- Binary Cross-Entropy

---

### 5. Функции активации

Реализованы следующие передаточные (активационные) функции:

- ReLU
- Sigmoid
- Tanh
- Softmax

---

### 6. Обучение модели

Процесс обучения реализован в компактной форме и требует минимального количества кода при сохранении гибкости настройки.

Пример:
```python
model.compile(
    optimizer=SGD(lr=0.01),
    loss="cross_entropy"
)

model.fit(X_train, y_train, epochs=10, batch_size=32)
```

## Примеры применения

## Примеры

### 1. Классификация Iris

Запуск:

```bash
python examples/iris_classification.py
```
**Этот пример демонстрирует:**

- Многоклассовую классификацию с `Softmax` и `CategoricalCrossEntropy`.
- Отслеживание метрик на валидационной выборке.
- Оптимизатор `Adam` с ограничением градиента.

### 2. Регрессия синусоиды

Запуск:

```bash
python examples/sine_regression.py
```

**Этот пример демонстрирует:**

- Регрессию на синтетическом датасете.
- Оптимизатор `MomentumSGD`.
- Метрики `mse` и `mae`.

### 3. Детектор котиков по изображениям

Запуск:

```bash
python examples/cat_detector.py --data-dir data/cats_binary --image-size 32 --epochs 40
```

**Этот пример демонстрирует:**

- Загрузку изображений из папок.
- Бинарную классификацию картинок при помощи полносвязной сети.

После обучения скрипт сохраняет:

- веса модели, по умолчанию в `artifacts/cat_detector_weights.npz`
- метаданные модели, по умолчанию в `artifacts/cat_detector_meta.json`

Чтобы заново обучить модель, достаточно снова запустить `cat_detector.py`: веса будут пересчитаны с нуля и сохранены заново.

Чтобы проверить новую картинку уже обученной моделью:

```bash
python examples/predict_cat.py --image path/to/new_image.jpg
```

### 4. Предсказание выживания пассажиров Титаника

**Запуск:**

```bash
python examples/titanic_classification.py --zip-path data/titanic.zip
```

**Этот пример демонстрирует:**

* Бинарную классификацию на табличных данных из датасета Titanic.
* Ручную предобработку смешанных признаков: пол, возраст, стоимость билета, класс, порт посадки, размер семьи, наличие каюты и титул пассажира.
* Стратифицированное разделение на обучающую и валидационную выборки.
* Обучение нескольких моделей с разными начальными значениями случайных сидов и выбор лучшей из них.
* Сравнение алгоритмов `SGD`, `MomentumSGD` и `Adam` на одних и тех же признаках.
* Генерацию визуальных артефактов для презентации и анализа.

**Скрипт создаёт:**

* веса модели в `artifacts/titanic_weights.npz`
* метаданные предобработки в `artifacts/titanic_meta.json`
* SVG-отчёт в `artifacts/titanic_report.svg`
* PNG-графики в `artifacts/titanic_charts/`
* анимированный GIF процесса обучения в `artifacts/titanic_training.gif`

**Почему этот пример полезен:**

* Он показывает, что `MiniFlow` подходит не только для простых числовых задач, но и для реальных задач машинного обучения на табличных данных.
* Он демонстрирует полный pipeline: загрузка сырых CSV-данных, инженерия признаков, обучение модели, оценка качества и генерация отчётов.
